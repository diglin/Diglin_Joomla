### Diglin Joomla

Magento module integrating Joomla modules into Magento with the Jfusion project.

### Compatibility

- Joomla 1.5.x
- Magento 1.x

### Requirements

- Magento 1.x
- Joomla 1.5.x
- Jfusion plugins for Joomla
